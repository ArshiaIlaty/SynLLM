## Data Description

The cirrhosis dataset is a part of the UCI machine learning repository and contains data about the progression of cirrhosis. Cirrhosis is a disease in which healthy liver tissue is replaced with scar tissue, eventually preventing the liver from functioning properly.

The data is a subset of a larger dataset and contains 301 records of patients with cirrhosis. There are 22 variables in the data set, including patient ID, follow-up days, status, treatment type, age, sex, ascites, hepatomegaly, spiders, edema, bilirubin, cholesterol, albumin, copper, alkaline phosphatase, aspartate aminotransferase, tryglycerides, platelets, prothrombin, and histologic stage.

The data is divided into two classes: censored and transplant or death. The data is balanced with 50% of the records being censored and 50% being transplant or death.

The data is publicly available on the UCI machine learning repository and is used in many machine learning competitions.

16. SGOT: aspartate aminotransferase (U/L)  
17. Tryglicerides: mg/dL  
18. Platelets: count (×1000/mm³)  
19. Prothrombin: seconds  
20. Stage: histologic stage (1-4)

## Usage

1. Install the package `synthetic-data-generators`.

    ```bash
    pip install synthetic-data-generators
    ```

2. Install the cirrhosis package.

    ```bash
    pip install cirrhosis
    ```

3. Run the program.

    ```bash
    python cirrhosis.py
    ```

    This will generate 1000 records.
