```python
def create_record(df, status, drug, age, sex, ascites, hepatomegaly, spiders, edema, bilirubin, cholesterol, albumin, copper, alk_phos, sgot, tryglicerides, platelets, prothrombin, stage):
    id = random.randint(1, 1000)
    record = np.zeros(21)
    record[0] = id
    record[1] = df.N_Days.mean()
    record[2] = status
    record[3] = drug
    record[4] = age
    record[5] = df.Sex.value_counts().index[np.random.choice(df.Sex.value_counts().index)]
    record[6] = ascites
    record[7] = hepatomegaly
    record[8] = spiders
    record[9] = edema
    record[10] = bilirubin
    record[11] = cholesterol
    record[12] = albumin
    record[13] = copper
    record[14] = alk_phos
    record[15] = sgot
    record[16] = tryglicerides
    record[17] = platelets
    record[18] = prothrombin
    record[19] = stage
    return record
```

### Data Format

The data file must be a CSV file with 17 columns. The columns are as follows:

- ID: Unique identifier for the record.
- N_Days: Number of days since diagnosis.
- Status: Status of the patient, D, C, or CL.
- Drug: D-penicillamine or Placebo.
- Age: Age of the patient.
- Sex: Sex of the patient, F or M.
- Ascites: Ascites status, Y or N.
- Hepatomegaly: Hepatomegaly status, Y or N.
- Spiders: Spider angiomas status, Y or N.
- Edema: Edema status, S (significant), Y (yes), or N (no).
- Bilirubin: Bilirubin level, mg/dL.
- Cholesterol: Cholesterol level, mg/dL.
- Albumin: Albumin level, g/dL.
- Copper: Copper level, µg/dL.
- Alk_Phos: Alkaline phosphatase level, U/L.
- SGOT: Serum glutamic-oxaloacetic transaminase level, U/L.
- Tryglicerides: Tryglycerides level, mg/dL.
- Platelets: Platelet count, x1000/mm³.
- Prothrombin: Prothrombin time, seconds.
- Stage: Stage of cirrhosis, 1 to 4.

```python
import cirrhosis
```
